ProtoxZF

Conjunto de código para formar um arquitetura robusta para inicialização de novos projetos que utilizam Zend Framework versão 1.X. Atualmente este projeto não é mais atualizado.

  • Página do projeto: ProtoxZF

Thinbox

Aplicativo para dispositivos móveis ainda em desenvolvimento escrito em Django e JQuery Mobile baseado em SaaS e feito para controle do RouterOS, sistema operacional de roteamento da Mikrotik baseado em Linux e bastante utilizado provedores de internet a rádio. Este aplicativo é composto por um módulo desktop e um mobile visando funcionamento em Android e iOS.

  • Página do projeto: Thinbox, Thinbox Android
